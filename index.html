Arrays Unidimensionals (1D)

Què és un array 1D? És una estructura de dades que emmagatzema elements en una sola dimensió (com una llista). Cada element té un índex que comença en 0.
// Declaració i inicialització d'un array int[] numeros = {10, 25, 30, 45, 50}; // Accés a elements int primerElement = numeros[0]; // primerElement = 10 int tercerElement = numeros[2]; // tercerElement = 30 // Longitud de l'array int longitud = numeros.length; // longitud = 5

Arrays Bidimensionals (2D - Matrius)

Què és un array 2D? És una estructura que emmagatzema elements en files i columnes (com una taula). S'accedeix mitjançant dos índexs: [fila][columna].
// Declaració i inicialització d'una matriu int[][] matriu = { {1, 2, 3}, {4, 5, 6}, {7, 8, 9} }; // Accés a elements int element = matriu[1][2]; // element = 6 (fila 1, columna 2) // Dimensions int files = matriu.length; // files = 3 int columnes = matriu[0].length; // columnes = 3

Funcions SENSE Paràmetres

Què són? Funcions que NO reben dades d'entrada. Executen un bloc de codi i poden o no retornar un valor.
// Exemple 1: Funció que retorna un String public static String saludar() { return "Hola Món!"; } // Exemple 2: Funció que retorna un número aleatori public static int obtenirNumeroAleatori() { return (int)(Math.random() * 100); } // Crida des del main: public static void main(String[] args) { String missatge = saludar(); System.out.println(missatge); // Mostra: Hola Món! int numero = obtenirNumeroAleatori(); System.out.println("Número: " + numero); }

Funcions AMB Paràmetres

Què són? Funcions que SÍ reben dades d'entrada (paràmetres). Processen aquestes dades i poden retornar un resultat.
// Exemple 1: Funció que suma dos números public static int sumar(int a, int b) { return a + b; } // Exemple 2: Funció que saluda una persona public static String saludarPersona(String nom) { return "Hola, " + nom + "!"; } // Crida des del main: public static void main(String[] args) { int resultat = sumar(5, 3); System.out.println("5 + 3 = " + resultat); // Mostra: 5 + 3 = 8 String salutacio = saludarPersona("Maria"); System.out.println(salutacio); // Mostra: Hola, Maria! }

Funcions RECURSIVES

Què són? Funcions que es criden a si mateixes per resoldre un problema dividint-lo en subproblemes més petits. Necessiten un cas base per evitar bucles infinits.
Components d'una funció recursiva:
1. Cas base: Condició que deté la recursivitat
2. Cas recursiu: La funció es crida a si mateixa amb un problema més petit
// Exemple: Factorial recursiu public static int factorial(int n) { if (n == 0 || n == 1) { return 1; // Cas base } return n * factorial(n - 1); // Cas recursiu } // Crida des del main: public static void main(String[] args) { int resultat = factorial(5); System.out.println("5! = " + resultat); // Mostra: 5! = 120 }

Funció Recursiva: factorial(n)

El factorial de n (n!) és el producte de tots els nombres enters positius menors o iguals a n.
Exemple: 5! = 5 × 4 × 3 × 2 × 1 = 120

Com Cridar Funcions des del Main

Concepte clau: El mètode main és el punt d'entrada del programa. Des d'aquí cridem totes les altres funcions que hem creat.

Exemple 1: Funció SENSE Paràmetres

public class Programa { // Funció que volem cridar public static String saludar() { return "Hola Món!"; } // Mètode main - punt d'entrada public static void main(String[] args) { // Cridem la funció i guardem el resultat String missatge = saludar(); System.out.println(missatge); // Mostra: Hola Món! } }

Exemple 2: Funció AMB Paràmetres

public class Programa { // Funció que rep dos paràmetres public static int sumar(int a, int b) { return a + b; } // Mètode main public static void main(String[] args) { // Cridem la funció passant dos valors int resultat = sumar(5, 3); System.out.println("Resultat: " + resultat); // Mostra: Resultat: 8 } }

Exemple 3: Funció RECURSIVA

public class Programa { // Funció recursiva public static int factorial(int n) { if (n == 0 || n == 1) { return 1; // Cas base } return n * factorial(n - 1); // Es crida a si mateixa } // Mètode main public static void main(String[] args) { int resultat = factorial(3); System.out.println("3! = " + resultat); // Mostra: 3! = 6 } }
Punts clau a recordar:
1. El main() és el punt d'inici del programa
2. Les funcions es criden escrivint el seu nom i passant els paràmetres (si en té)
3. El valor que retorna la funció es pot guardar en una variable
4. Les funcions recursives es criden a si mateixes fins arribar al cas base

Arrays 2D

✓ Dues dimensions (taula)

✓ Índex: [i][j]

✓ No reben entrada

✓ Sintaxi: funcio()

Funcions amb paràmetres

✓ Reben entrada

✓ Sintaxi: funcio(param)

✓ Exemple: suma(a, b)
